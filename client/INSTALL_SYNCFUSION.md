# Syncfusion Installation Command

## Quick Install Command

Copy and paste this single command to install all required Syncfusion packages:

```bash
npm install @syncfusion/ej2-base@^31.1.20 @syncfusion/ej2-react-buttons@^31.2.12 @syncfusion/ej2-react-calendars@^31.1.21 @syncfusion/ej2-react-dropdowns@^31.1.20 @syncfusion/ej2-react-grids@^31.1.21 @syncfusion/ej2-react-interactive-chat@^31.2.5 @syncfusion/ej2-react-navigations@^31.1.20 @syncfusion/ej2-react-notifications@^31.2.2 @syncfusion/ej2-react-popups@^31.2.12 @syncfusion/ej2-react-richtexteditor@^31.1.21
```

## What This Installs

This will install all 10 Syncfusion packages needed for the project:

1. `@syncfusion/ej2-base` - Base package (required for all components)
2. `@syncfusion/ej2-react-buttons` - Button components
3. `@syncfusion/ej2-react-grids` - Data grid/table components
4. `@syncfusion/ej2-react-popups` - Dialog/modal components
5. `@syncfusion/ej2-react-notifications` - Toast notification components
6. `@syncfusion/ej2-react-dropdowns` - Dropdown/select components
7. `@syncfusion/ej2-react-calendars` - Date/time picker components
8. `@syncfusion/ej2-react-richtexteditor` - Rich text editor components
9. `@syncfusion/ej2-react-navigations` - Tab navigation components
10. `@syncfusion/ej2-react-interactive-chat` - Chat UI components

## Notes

- Make sure you're in the `ability_V2/client` directory before running this command
- The license key should be set in `.env` file as `REACT_APP_SYNCFUSION_LICENSE`
- All CSS styles are already imported in `src/index.js`
